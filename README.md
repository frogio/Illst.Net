# Illust.Net
멀티캠퍼스 웹 인터페이스 프로젝트

Windows 10, Django 4.1.1, SQLite3 기반으로 작성됨

이미지 스크랩 소셜 네트워크 서비스인 pinterest를 모방하여 만든 일러스트 공유 사이트 입니다.
